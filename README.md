# GitHub Pages Template (Researcher Style)

This is a lightweight Jekyll template based on the `ankitsultana/researcher` theme, customized to match the style of https://parkchanjun.github.io/ (font stack, colors, and layout).

## Quick Start

1. Edit `_config.yml` (site title, URL, nav links).
2. Edit `index.md` (your content).
3. Add `profile.jpg` and `cv.pdf` to the project root.
4. Push the repo to GitHub and enable Pages for the `main` branch.

## Customize

- Accent color and image size: `_sass/vars.scss`
- Layout and meta tags: `_layouts/default.html`
- Typography and spacing: `_sass/typography.scss`
